## Claude (Anthropic)

### Claude Code CLI

- **GitHub Repository**: https://github.com/anthropics/claude-code
- **Official Website**: https://www.anthropic.com/claude-code
- **Documentation**: https://docs.claude.com/en/docs/claude-code
- **GitHub Actions**: https://github.com/anthropics/claude-code-action
- **License**: Proprietary
- **Description**: Agentic coding tool that lives in your terminal, understands your codebase, and helps you code faster through natural language commands

### Claude Desktop

- **Official Download**: https://claude.com/download
- **Documentation**: https://support.claude.com/en/articles/10065433-installing-claude-desktop
- **Desktop Extensions**: https://www.anthropic.com/engineering/desktop-extensions
- **Platforms**: macOS 11.0+, Windows 10+
- **Description**: Native desktop application with MCP support and desktop extensions

### Claude for VS Code

- **VS Code Marketplace**: https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code
- **GitHub Repository**: https://github.com/anthropics/claude-code (VS Code integration)
- **Documentation**: https://docs.anthropic.com/en/docs/claude-code/ide-integrations
- **Status**: Beta
- **Description**: Official VS Code extension for Claude Code with real-time diff viewing

## Model Context Protocol (MCP)

### Official MCP Resources

- **Official Website**: https://modelcontextprotocol.io/
- **Documentation**: https://modelcontextprotocol.io/docs
- **GitHub Repository**: https://github.com/modelcontextprotocol
- **Server Repository**: https://github.com/modelcontextprotocol/servers
- **License**: Varies by server (most are MIT/Apache 2.0)
- **Description**: Standardized protocol for connecting AI assistants to external data sources and tools

### Popular MCP Servers

- **Filesystem**: `@modelcontextprotocol/server-filesystem`
- **GitHub**: `@modelcontextprotocol/server-github`
- **Brave Search**: `@modelcontextprotocol/server-brave-search`
- **Sequential Thinking**: `@modelcontextprotocol/server-sequential-thinking`
- **Memory**: `@modelcontextprotocol/server-memory`
- **Puppeteer**: `@modelcontextprotocol/server-puppeteer`
- **PostgreSQL**: `@modelcontextprotocol/server-postgres`
- **Slack**: `@modelcontextprotocol/server-slack`
